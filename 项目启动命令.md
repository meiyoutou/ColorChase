# 服务器启停命令

## 启动服务器

```powershell
cd D:\桌面\Trae临时文件\ColorChase
& ".venv312\Scripts\python.exe" main.py
```

## 重启服务器

```powershell
# 1. 停止旧进程（按端口查找并杀掉）
Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ -Force -ErrorAction SilentlyContinue }

# 2. 启动新进程
cd D:\桌面\Trae临时文件\ColorChase
& ".venv312\Scripts\python.exe" main.py
```

## 单行重启命令（可直接复制）

```powershell
Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ -Force -ErrorAction SilentlyContinue }; cd D:\桌面\Trae临时文件\ColorChase; & ".venv312\Scripts\python.exe" main.py
```

## 停止服务器

```powershell
Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ -Force -ErrorAction SilentlyContinue }
```

## 关键信息

| 项目 | 值 |
|---|---|
| 虚拟环境 | `.venv312`（项目根目录） |
| Python 解释器 | `.venv312\Scripts\python.exe` |
| 启动文件 | `main.py` |
| 监听端口 | 8000 |
| 访问地址 | http://127.0.0.1:8000 |

## 备注

- 启动命令在终端会**前台运行**（占用终端），按 `Ctrl+C` 可停止
- 重启命令先杀掉占用 8000 端口的进程，再启动新进程
- 若端口被占用且无法启动，先手动执行停止命令，再执行启动命令
